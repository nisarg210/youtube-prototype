## Getting Started

### Backend

1. Navigate to the backend directory:

   ```bash
   cd path/to/each-microservice-backend
   ```

2. Install the dependencies:

   ```bash
   npm install
   ```

3. Start the backend service:

   ```bash
   npm start
   ```

### Frontend

1. Navigate to the frontend directory:

   ```bash
   cd path/to/frontend
   ```

2. Install the dependencies:

   ```bash
   npm install
   ```

3. Start the frontend service:

   ```bash
   npm start
   ```
